# legendary-waddle
아주대학교 창의소프트웨어입문 파이썬 과제 (영화 예매 프로그램)

## Prerequisite
* `numpy` ~= 1.21.4
* `pandas` ~= 1.3.4
* `questionary` ~= 1.10.0
* `pyfiglet` ~= 0.8.post1

## Installation
```Shell
git clone https://github.com/KyoungsueKim/legendary-waddle
cd legendary-waddle
pip install -r requirements.txt
```

## Run
```Shell
python3 legendary-waddle/main.py
```
